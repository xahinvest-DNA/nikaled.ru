# Video Pipeline

Эта папка хранит материалы для ролика в гибридном пайплайне: сначала сильные keyframes, потом короткие видеофрагменты, потом сборка в один вертикальный рекламный ролик.

## Почему именно гибридный путь

На этой машине не хватает ресурсов для современных локальных video-to-video и text-to-video моделей.

- GPU: `NVIDIA GeForce 9800 GT`
- Dedicated VRAM: около `482 MB`
- RAM: `8 GB`

Для моделей уровня Wan, LTX или похожих этого недостаточно.

## Рекомендуемый workflow

1. Сгенерировать 4-6 сильных опорных кадров.
2. Анимировать каждый кадр отдельно как клип на 3-5 секунд.
3. Склеить клипы в вертикальный ролик на 15-20 секунд.
4. Поверх добавить субтитры и озвучку.

## Состав папки

- `shotlist.md` — структура сцен и промпты по кадрам
- `voiceover.txt` — текст озвучки
- `subtitles.srt` — субтитры с таймингами

## Практический следующий шаг

Самый быстрый путь на этой машине:

1. Сгенерировать консистентные keyframes по этим промптам.
2. Анимировать кадры в облачном видео-инструменте или на более сильной машине.
3. При необходимости позже собрать всё локально через `ffmpeg`.

## Примечания

- Лучше 4-5 коротких сцен, чем один длинный ролик.
- Кот должен быть одинаковым во всех сценах: пушистый рыжий, реалистичная анатомия, серьёзное выражение.
- Каждую сцену лучше генерировать отдельно, так легче удержать стиль и мимику.
